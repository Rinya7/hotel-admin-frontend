# Assets (Статичні ресурси)

## Призначення папки

Папка містить статичні ресурси додатку - зображення, стилі, шрифти та інші файли, які використовуються в інтерфейсі.

## Структура

```
src/assets/
├── *.png, *.jpg       # Зображення (логотипи, фони)
├── tailwind.css       # Tailwind CSS стилі
└── vue.svg            # SVG іконки
```

## Файли

### Зображення

**Логотипи:**
- `logo_*.png` - різні варіанти логотипів
- `logo_HoteLoste.png` - основний логотип
- `logo_white.png`, `logo_black.png` - логотипи для різних тем

**Фони:**
- `bg_image.jpg`, `bg_picktures.jpg` - фонові зображення

**Інші:**
- `vue.svg` - SVG іконка Vue

### tailwind.css

**Призначення:** Головний файл стилів Tailwind CSS.

**Що містить:**

- Імпорти Tailwind директив
- Кастомні стилі
- Глобальні стилі додатку

**Використання:**

- Імпортується в `main.ts`
- Використовується по всьому додатку через Tailwind класи

## Принципи

- **Оптимізація** - зображення мають бути оптимізовані
- **Організація** - логічне іменування файлів
- **Розміри** - використовувати відповідні формати (PNG для логотипів, JPG для фото)
