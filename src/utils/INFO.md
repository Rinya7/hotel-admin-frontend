# Utils (Допоміжні функції)

## Призначення папки

Папка містить утилітарні функції для форматування даних, роботи з датами, обробки рядків та інших допоміжних операцій.

## Структура

```
src/utils/
├── date.ts            # Функції для роботи з датами
├── formatAddress.ts   # Форматування адреси
└── formatPhone.ts     # Форматування телефону
```

## Файли

### date.ts

**Призначення:** Функції для форматування та роботи з датами.

**Що містить:**

- Функції форматування дат для відображення
- Конвертація між форматами
- Робота з часовими зонами

**Використання:**

```typescript
import { formatDate, formatDateTime } from "@/utils/date";

const formatted = formatDate(new Date());
```

### formatAddress.ts

**Призначення:** Форматування детальної адреси готелю в рядок для відображення.

**Що містить:**

1. **formatHotelAddress(hotel):**
   - Форматує адресу з полів: street, buildingNumber, apartmentNumber, province, postalCode, country
   - Повертає відформатований рядок або "—" якщо адреси немає
   - Формат: "вулиця, номер, кв. X, провінція, індекс, країна"

2. **getAddressSearchString(hotel):**
   - Отримує рядок для пошуку (всі частини адреси)
   - Використовується для фільтрації та пошуку

**Використання:**

```typescript
import { formatHotelAddress } from "@/utils/formatAddress";

const address = formatHotelAddress(hotel);
// Результат: "Via Roma, 10, кв. 5, Milano, 20100, Italy"
```

### formatPhone.ts

**Призначення:** Форматування номера телефону з коду країни та номера.

**Що містить:**

- `formatPhoneNumber(countryCode, phoneNumber)` - форматує телефон
- Обробка різних форматів
- Додавання коду країни з "+"

**Використання:**

```typescript
import { formatPhoneNumber } from "@/utils/formatPhone";

const phone = formatPhoneNumber("+39", "123456789");
// Результат: "+39 123456789"
```

## Принципи

- **Чисті функції** - функції не мають side effects
- **Типізація** - всі функції типізовані
- **Перевикористання** - функції можна використовувати в різних місцях
- **Тестованість** - функції легко тестувати
