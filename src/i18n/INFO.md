# i18n (Інтернаціоналізація)

## Призначення папки

Папка містить конфігурацію інтернаціоналізації (i18n) - переклади інтерфейсу на різні мови. Підтримує українську, англійську, італійську та німецьку мови.

## Структура

```
src/i18n/
├── index.ts          # Конфігурація i18n
└── locales/          # Файли перекладів
    ├── uk.ts         # Українська
    ├── en.ts         # Англійська
    ├── it.ts         # Італійська
    └── de.ts         # Німецька
```

## Файли

### index.ts

**Призначення:** Конфігурація Vue I18n плагіна.

**Що робить:**

- Налаштовує Vue I18n
- Реєструє локалі (uk, en, it, de)
- Встановлює мову за замовчуванням
- Налаштовує fallback мову

**Експортує:**

- `i18n` - налаштований екземпляр Vue I18n

**Використання:**

```typescript
import { i18n } from "./i18n";

app.use(i18n);
```

### locales/uk.ts, en.ts, it.ts, de.ts

**Призначення:** Файли перекладів для кожної мови.

**Структура:**

```typescript
export default {
  common: {
    save: "Зберегти",
    cancel: "Скасувати",
    // ...
  },
  rooms: {
    title: "Кімнати",
    // ...
  },
  // ...
};
```

**Організація:**

- `common` - загальні переклади (кнопки, дії)
- `rooms` - переклади для кімнат
- `stays` - переклади для проживань
- `auth` - переклади для авторизації
- `dashboard` - переклади для dashboard
- `superadmin` - переклади для superadmin
- `notifications` - переклади для сповіщень

**Використання:**

```typescript
import { useI18n } from "vue-i18n";

const { t } = useI18n();

const text = t("common.save");
```

## Підтримувані мови

- **uk** - Українська (за замовчуванням)
- **en** - English
- **it** - Italiano
- **de** - Deutsch

## Принципи

- **Ключі** - використовувати описові ключі (наприклад, `rooms.create.title`)
- **Організація** - переклади організовані за модулями
- **Синхронізація** - всі мови мають однакові ключі
- **Fallback** - якщо перекладу немає, використовується fallback мова

## Додавання нового перекладу

1. Визначити ключ (наприклад, `rooms.newFeature.title`)
2. Додати переклад у всі файли локалей
3. Використати в компоненті через `t("rooms.newFeature.title")`
