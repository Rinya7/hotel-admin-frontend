# Constants (Константи)

## Призначення папки

Папка містить константи та значення за замовчуванням, які використовуються в додатку. Синхронізовано з `hotel-backend/src/constants/`.

## Структура

```
src/constants/
├── defaults.ts           # Значення за замовчуванням
├── countries.ts          # Список країн
├── italianProvinces.ts  # Список італійських провінцій
└── README.md            # Документація
```

## Файли

### defaults.ts

**Призначення:** Значення за замовчуванням для системи (години заїзду/виїзду, логотипи).

**Що містить:**

- `DEFAULT_CHECK_IN_HOUR` - година заїзду за замовчуванням (14:00)
- `DEFAULT_CHECK_OUT_HOUR` - година виїзду за замовчуванням (10:00)
- `DEFAULT_HOTEL_LOGO_VARIANTS` - варіанти логотипів
- `DEFAULT_HOTEL_LOGO_URL` - логотип за замовчуванням
- `getRandomHotelLogo()` - функція для випадкового логотипу

**Важливо:**

- Синхронізовано з `hotel-backend/src/constants/defaults.ts`
- Зміни мають бути в обох файлах

### countries.ts

**Призначення:** Список країн для випадаючого списку.

**Що містить:**

- Масив об'єктів з кодом країни та назвою
- Використовується в формах створення/редагування готелю

**Синхронізація:**

- Синхронізовано з `hotel-backend/src/constants/countries.ts`

### italianProvinces.ts

**Призначення:** Список італійських провінцій для випадаючого списку.

**Що містить:**

- Масив об'єктів з кодом провінції та назвою
- Використовується в формах створення/редагування готелю

**Синхронізація:**

- Синхронізовано з `hotel-backend/src/constants/italianProvinces.ts`

## Важливі моменти

- **Синхронізація** - константи мають бути синхронізовані з бекендом
- **Централізація** - всі значення за замовчуванням в одному місці
- **Легка зміна** - можна швидко змінити налаштування
